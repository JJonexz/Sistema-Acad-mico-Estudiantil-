# Sistema Académico Estudiantil de Promoción

## Descripción del Proyecto

El **Sistema Académico Estudiantil de Promoción** es una herramienta informática diseñada para analizar la situación académica de los estudiantes de nivel secundario.

El sistema permitirá registrar estudiantes, materias y calificaciones, y a partir de estos datos **determinar automáticamente la situación académica de cada estudiante** según las reglas del régimen educativo vigente.

Este proyecto se desarrolla como trabajo académico para la **Escuela Técnica N°1 "Raúl Scalabrini Ortiz"**.

## Objetivo del Sistema

El objetivo del sistema es **automatizar el análisis académico de los estudiantes**, permitiendo determinar su situación escolar de forma clara y rápida.

El sistema permitirá:

* Registrar estudiantes
* Registrar materias
* Registrar calificaciones
* Analizar información académica
* Determinar la situación académica de los estudiantes

---

## Funcionalidades principales

### Gestión de estudiantes

* Registrar estudiantes
* Editar información de estudiantes
* Visualizar lista de estudiantes

### Gestión de materias

* Registrar materias
* Asociar materias a estudiantes
* Visualizar materias registradas

### Gestión de calificaciones

* Registrar notas
* Modificar notas
* Visualizar historial de calificaciones

### Análisis académico

El sistema analizará las calificaciones para determinar:

* Materias aprobadas
* Materias pendientes
* Necesidad de intensificación
* Promoción al año siguiente
* Necesidad de recursar materias
